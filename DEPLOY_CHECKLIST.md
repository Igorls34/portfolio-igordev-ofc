# Checklist de Preparação para Deploy

Este é um guia para finalizar o portfólio e garantir que ele esteja 100% pronto para ser publicado. Marque cada item conforme for concluído.

---

## 🔴 Passos Críticos (Obrigatório)

-   [ ] **Finalizar a Seção de Projetos**
    -   [ ] Subir o código-fonte dos projetos para repositórios no GitHub.
    -   [ ] Atualizar o `index.html` com os links reais para os repositórios do GitHub.
    -   [ ] Escrever descrições claras e objetivas para cada projeto.

-   [ ] **Remover Comentários de Código**
    -   [ ] Apagar o comentário `<!-- TODO: ... -->` da seção de projetos no `index.html`.

---

## 🟡 Passos Recomendados (Melhorias Profissionais)

-   [ ] **Melhorar SEO (Otimização para Buscadores)**
    -   [ ] Adicionar a tag `<meta name="description" content="...">` no `<head>` do `index.html`.
    -   **Sugestão de conteúdo:** `<meta name="description" content="Desenvolvedor Freelancer especialista em criação de sites responsivos e automações com Python. Transformo suas necessidades em código funcional e eficiente.">`

-   [ ] **Otimizar Performance**
    -   [ ] Comprimir a imagem `avatar-1772253458668.png` usando uma ferramenta como [TinyPNG](https://tinypng.com/).
    -   [ ] Substituir a imagem antiga pela versão otimizada na pasta `assets/img/`.

-   [ ] **Organizar o Código**
    -   [ ] Mover o código JavaScript do formulário de contato (que está no final do `index.html`) para o arquivo `assets/js/script.js`.
    -   [ ] Garantir que a tag `<script src="./assets/js/script.js"></script>` permaneça no `index.html`.

-   [ ] **Limpar o Projeto**
    -   [ ] Apagar o arquivo `assets/css/style-two.css`.
    -   [ ] Apagar o arquivo `assets/css/style-three.css`.
    -   [ ] Apagar a imagem `assets/img/Sudharsan.jpg` (se não estiver em uso).

-   [ ] **Validação Final**
    -   [ ] Testar o formulário de contato mais uma vez para garantir o envio.
    -   [ ] Clicar em todos os links (lembredes sociais, WhatsApp, projetos) para confirmar que estão corretos.
    -   [ ] Abrir o site em um celular (ou no modo de desenvolvedor do navegador) para verificar se o layout responsivo está perfeito.

---

Com este checklist, você terá um caminho claro para um deploy de sucesso!
